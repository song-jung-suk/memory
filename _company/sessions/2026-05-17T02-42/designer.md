## 🎬 디자인 브리프: 유튜브 쇼츠 (Top 3)

### 1. 주제: 실패 방지 공식 공개 (불안감/손실 대비 통제감)

**🎯 목표 감정:** 불안감(Pain)에서 즉각적인 통제감(Gain)으로의 전환
**💡 핵심 컨셉:** **혼돈(Chaos) vs. 질서(Order)**

| 요소 | 상세 디자인 지침 | 적용 컬러 팔레트 (극단적 대비) | 레이아웃 컨셉 |
| :--- | :--- | :--- | :--- |
| **썸네일** | 화면이 중앙에서 분열되는 형태. 왼쪽은 어둡고 복잡한 그래프(손실), 오른쪽은 깨끗하고 정돈된 시스템 블록(해결책)으로 나뉘어 있음. 텍스트는 빨간색 경고와 녹색 해결책을 명확히 대비. | **Pain (좌측):** #FF4136 (강렬한 레드), #1A1A1A (짙은 블랙) / **Gain (우측):** #00BFFF (밝은 시안/스카이 블루), #FFFFFF (화이트) | 분할 레이아웃 (Split Screen). 왼쪽 50%는 어둡고 혼란스러운 패턴, 오른쪽 50%는 밝고 선명한 그리드 시스템. |
| **핵심 장면** | 1. **Pain:** 복잡하게 얽힌 데이터나 비용 항목들이 빠르게 스크롤되며 화면을 채움 (불안감 극대화). 2. **Transition:** 화면이 순간적으로 정지하며 '공식'이라는 단어가 중앙에 강렬하게 등장. 3. **Gain:** 깔끔한 공식(Formula) 블록이 차분하게 나타나며, 시청자가 명확한 경로를 보게 함. | **전환 효과:** 레드에서 블루로의 급격한 색상 전환 (Color Shift). | 동적 흐름 (Dynamic Flow). 혼란스러운 움직임에서 정적인 확정으로의 안정적인 이동. |
| **폰트** | Pain 부분: 고딕 계열, 굵고 각진 서체. Gain 부분: 산세리프 계열, 간결하고 명확한 서체. | | |

### 2. 주제: 시간 대비 기회 비용 (후회/탐욕 대비 실행)

**🎯 목표 감정:** 후회(과거의 낭비)에서 탐욕(미래의 이득)으로의 추진력
**💡 핵심 컨셉:** **느림(Loss) vs. 속도(Gain)**

| 요소 | 상세 디자인 지침 | 적용 컬러 팔레트 (극단적 대비) | 레이아웃 컨셉 |
| :--- | :--- | :--- | :--- |
| **썸네일** | 시계와 돈이 겹쳐지는 시각 효과. 왼쪽은 느리게 움직이는 모래시계(시간의 흐름/손실), 오른쪽은 폭발적으로 움직이는 화살표 또는 성장 그래프(기회/탐욕). | **Loss (좌측):** #8B0000 (짙은 버건디 레드), #4A4A4A (차콜 그레이) / **Gain (우측):** #FFD700 (골드), #006400 (딥 그린) | 대각선 분할 및 움직임 강조. 왼쪽 하단에서 오른쪽 상단으로 향하는 강력한 시각적 흐름(Directional Flow). |
| **핵심 장면** | 1. **Pain:** 느리고 무거운 노동의 순간을 슬로우 모션으로 표현하며, 화면에 시간의 압박감을 시각화 (손실감 강조). 2. **Transition:** 시간이 멈추거나 갑자기 가속되는 효과를 통해 '자동화' 시스템이 작동하는 장면 삽입. 3. **Gain:** 시스템이 자동으로 수익을 생성하는 모습을 빠르고 명확하게 보여주며, 돈이 기하급수적으로 증가하는 것을 시각화 (탐욕 자극). | **대비 강조:** 차가운 색(Loss)과 따뜻한 색(Gain)의 강렬한 대비. | 흐름 기반 레이아웃 (Flow Layout). 시간의 흐름을 따라 화면 전체를 채우는 역동적인 움직임. |
| **폰트** | Pain 부분: 굵고 느린 움직임을 표현하는 서체. Gain 부분: 날렵하고 속도감 있는 서체. | | |

### 3. 주제: 숨겨진 함정 공개 (불안감/위협 대비 안전)

**🎯 목표 감정:** 위협(Threat)에서 명확한 통제와 안전(Safety)으로의 안도감
**💡 핵심 컨셉:** **어둠(Chaos) vs. 빛(Control)**

| 요소 | 상세 디자인 지침 | 적용 컬러 팔레트 (극단적 대비) | 레이아웃 컨셉 |
| :--- | :--- | :--- | :--- |
| **썸네일** | 화면의 대부분을 어둡고 그림자가 드리운 상태로 설정하여 위협감을 조성. 중앙에 작은 빛(경고등 또는 돋보기)이 숨겨진 함정(Trap)을 비추는 형태. | **Chaos (배경):** #0A0A0A (무광 블랙), #36454F (어두운 청회색) / **Control (포인트):** #FFD700 (골드/옐로우), #FFFFFF (밝은 화이트) | 명암 대비 극대화 (High Contrast). 어둠 속에서 빛이 길을 안내하는 형태. |
| **핵심 장면** | 1. **Threat:** 화면에 불투명한 그림자나 흐릿한 데이터들이 위협적으로 움직이는 모습을 연출하여 숨겨진 위험(함정)의 존재를 암시. 2. **Transition:** 강력한 조명이 터지면서, 그 아래에 '안전 시스템'이라는 명확하고 견고한 구조가 드러남. 3. **Safety:** 모든 함정이 사라지고, 투명하고 안전한 보호막(Shield)이나 완벽한 차트가 등장하여 통제감을 부여함. | **대비 강조:** 암흑과 밝음의 극한 대비. 어둠 속에서 빛이 구원하는 느낌. | 중앙 집중형 (Central Focus). 위협 요소는 주변부에 배치하고, 해결책은 화면 중앙에 안정적으로 고정. |
| **폰트** | Threat 부분: 불안을 유발하는 굵고 긴장감 있는 서체. Safety 부분: 두껍고 신뢰감을 주는 산세리프 서체. | | |
